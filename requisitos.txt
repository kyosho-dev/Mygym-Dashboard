Requisitos do Sistema:


[Cadastro de Clientes]:

Os clientes devem ser registrados com informações como nome, idade, sexo, endereço e telefone. OK


[Cadastro de Treinadores]:

Os treinadores devem ser registrados com nome, especialização, experiência e número de registro.  OK


[Listagem de Clientes e Treinadores]:

O sistema deve exibir uma lista de todos os clientes e treinadores cadastrados. OK

O usuário deve poder atualizar informações de clientes e treinadores. OK


[Exclusão de Registros]:

Deve ser possível excluir clientes e treinadores. OK


[Armazenamento de Dados]:

Os dados devem ser persistidos em um banco de dados para recuperação posterior. OK
